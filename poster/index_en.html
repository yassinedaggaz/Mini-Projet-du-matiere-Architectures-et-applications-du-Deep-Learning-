Computer Hardware Image Classification
Transfer Learning with ResNet18 — Mini Project
Student · Module: Architectures & Applications of Deep Learning · May 2026

Abstract

This project implements a complete pipeline for multi-class image classification of computer hardware images (keyboard, laptop, monitor, mouse) using transfer learning. The goal is to build a usable classifier, evaluate its performance and provide tools for inference and visualization.

Dataset

Data organized using PyTorch ImageFolder standard. Split used (default): train/val/test (70/15/15).

  • train: 43 images
  • val: 8 images
  • test: 12 images

Method

ResNet18 pre-trained on ImageNet. The final fully-connected layer was replaced for 4 classes. Two modes supported:

  • Transfer learning: freeze backbone, train FC
  • Fine-tuning (optional): unfreeze layer4 + train FC

Preprocessing & Augmentation

  • Train: RandomResizedCrop(224), RandomHorizontalFlip, ColorJitter
  • Val/Test: Resize(224,224), normalization (ImageNet mean/std)

Code & Tools

Key scripts: src/train.py, src/evaluate.py, src/predict.py, src/gui.py
